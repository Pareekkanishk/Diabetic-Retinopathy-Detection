About This System

System Overview

An AI-powered system using transfer learning to analyze retinal images from the APTOS dataset and detect diabetic retinopathy. It provides rapid, accurate risk assessments, helping healthcare professionals diagnose the condition at an early stage.

Technology Stack

Backend: Flask, Python 3.9+

ML Models: InceptionV3, ResNet50

Image Processing: TensorFlow, OpenCV, CLAHE preprocessing

Frontend: HTML, CSS, JavaScript

Deployment: Docker
